The Legend of Zelda (NES): The Eagle + Custom Level

Built using C#, Unity, and GIMP, this project features a recreation of the classic NES game "The Legend of Zelda" with a focus on "The Eagle" dungeon and includes an exciting custom level.

Experience the nostalgia of the original game while enjoying new challenges in the custom-designed level.

My primary area of development revolved around the creation of enemies: movement, animation, health, and damage functionality. I also contributed to the precise recreation of weapons, collectibles, sprites, and custom level mechanics.

Space Invaders Remake

A modern take on the classic arcade game Space Invaders, developed using JavaScript and HTML5 Canvas.

This remake features updated graphics, smooth animations, and new power-ups while maintaining the addictive gameplay of the original.

I implemented the core game mechanics, including player movement, enemy AI, collision detection, and scoring system. The game also includes responsive design for various screen sizes.
